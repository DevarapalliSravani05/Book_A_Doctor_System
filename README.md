# 🏥 Doctor Appointment System
A modern and responsive **Doctor Appointment Booking System** built using the **MERN Stack**. This application enables patients to browse doctors, book appointments, and manage their bookings through an intuitive interface. It also includes an admin panel for managing doctors and appointments.
## 🚀 Features
### 👤 User
- User Registration & Login (JWT Authentication)
- Browse Doctors
- Search & Filter Doctors
- View Doctor Details
- Book Appointments
- View My Appointments
- Responsive UI

### 👨‍⚕️ Doctor
- Doctor Profiles
- Specialization
- Experience
- Consultation Fee
- Availability Status

### 🛠 Admin
- Admin Login
- Manage Doctors
- Manage Appointments
- Upload Doctor Images using Cloudinary

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/doctor-appointment-system.git
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
## 🔑 Environment Variables
Create a `.env` file inside the backend folder.
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
